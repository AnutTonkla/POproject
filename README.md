Math Battle
  เป็นเกมแนวสวมบทบาท ที่ใช้การคำนวณคณิตศาสตร์เป็นระะบบการต่อสู้หลักโดยใช้ TypeScript เป็นภาษาหลักในระบบต่างๆ 

main Features
  - ระบบสุ่มโจทย์คณิตศาสตร์แบบ Real-time ตามประเภทตัวละคร
  - ความถูกต้องและความเร็วของคำตอบมีผลต่อการต่อสู้

Upcoming Features
  - เวลาในการตอบจะขึ้นอยู่กับความต่างของความเร็วของ Hero และ monster
  - เลือกโจมตี หรือ ป้องกัน
  - มีการใช้ item
  - มีการเก็บ score 
  - monster จะมาเรื่อยๆจนกว่า Hero จะพ่ายแพ้

Tech Stack
  Language: TypeScript
  Frontend: HTML, CSS

How to Play
  - เลือกอาชีพ
  - คำนวณโจทย์ที่ปรากฏบนหน้าจอแล้วพิมพ์คำตอบ
  - กด Enter หรือปุ่ม Submit เพื่อ Action
